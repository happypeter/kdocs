>>>refer to apache doc/A.A.A---http://httpd.apache.org/docs/2.0/howto/auth.html#related
===below is the easist way to do auth, nothing to do with mysql
1. htpasswd -c /usr/local/apache/passwd/passwords peter
2. Now added the below stuff in conf.d/auth_mysql.conf
/*
AuthType Basic
AuthName "Restricted Files"
AuthUserFile /usr/local/apache/passwd/passwords
require user peter 
*/
3. works!

LoadModule mysql_auth_module modules/mod_auth_mysql.so
