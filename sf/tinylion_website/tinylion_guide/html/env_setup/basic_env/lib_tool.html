Compile Libs and Tools

  1. we need libqte.so.2.3.7

  2. we need tmake and progen

  3. we need moc and uic

  4. we need designer

  5. we need ...

Thanks to God, we do not have to compile them one by one
the package---ipaq-qtopia.tgz from Guangzhou FriendlyARM CO. Ltd. has them all Click to download
All we need is to follow these steps:

  1. Extract

  2. $ tar zxvf ...

  3. change /ipaq-qtopia/qtopia/src/qt/qconfig-qpe.h
    comment out
    #define QT_NO_TRANSFORMATIONS

  4. change set-env
    TMAKEPATH=$TMAKEDIR/lib/qws/linux-generic-g++
    into
    TMAKEPATH=$TMAKEDIR/lib/qws/linux-ipaq-g++

  5. $. set-env

  6. $cd $TMAKEPATH
    change tmake.conf
    LINK=...gcc------>g++
