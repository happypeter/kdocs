Tinylion

Jan 1, 2009

is a embedded linux project founded by Peter King. It contains open source hardware as well as software. The software contains driver and user_app.

  1. Hardware
    The hardware platform is based on S3C2410 ARM chip.
  2. Driver
    The driver is for sensor+adc+spi.
  3. User app
    The user application part, it is mainly a data analysis software targeted for scientific use. Tata is basicly the PC version of this part.
