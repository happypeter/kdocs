Cross Compiler Installation

  1. here we need arm-linux-gcc 3.3.2 Click to down load

  2. Maybe other versions will work as well, but it is just maybe, I need to stress ^_^

  3. Extact the package to install it in /usr/local/

  4. $ tar zxvf srm-linux-gcc-3.3.2.tgz -C /

  5. export the location of the tool, so that bash can find it

  6. $ export PATH=/usr/local/arm/3.3.2/bin/:$PATH

  7. try to write the above line into a boot script file if you do not want to type it anytime you colse the current terminal

  8. Check if it is well installed

  9. $ arm-linux-gcc -v

  10. Now the output is '3.3.2', that means everything is done
