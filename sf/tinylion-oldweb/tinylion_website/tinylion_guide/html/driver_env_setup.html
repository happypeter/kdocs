tar yc kernel into rh9,(no 'make' 'make install', nothing)

The kernel here is the one you are using on target platform
For tinylion it is s3c2410-linux-2.6.11.1.tar.bz2 (from Yctek disk)

change its name to s3c-2.6

copy the led driver in fc4 into rh9

use arm-linux-gcc-3.4.1
Note: other version can not garantee the job

issue make and a led.ko is obtianed.

that's all.

the same steps works on ubuntu

the Makefile in use is like this


KERNELDIR=/s3c-2.6/  #where the arm linux source is
PWD := $(shell pwd)
CROSS_COMPILE=arm-linux-
CC= $(CROSS_COMPILE)gcc
obj-m := spi.o 
modules:
	$(MAKE) -C $(KERNELDIR) M=$(PWD) modules
clean:
	rm  *mod* spi.o
