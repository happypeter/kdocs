Install Redhat9

  1. As far as I know, there is only one version of Redhat9.0

  2. It includes three isos, namely Shrike 1, 2, 3

  3. Start the installaion with many 'OK' 'Yes' and 'use default'

  4. until you are asked whether you want to install developer set of software or to 'customize'

  5. Please choose 'customize' here and choose 'everything' which means you want all the software installed

  6. Follow this with many defaults until the end of the installaion
